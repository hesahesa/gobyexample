Go dengan Contoh

Go adalah sebuah bahasa pemrograman open source yang dirancang untuk membuat software simpel, cepat, dan dapat diandalkan.

Go dengan Contoh adalah pengenalan praktik untuk Go dengan menggunakan contoh program dengan komentar. Silakan cek contoh pertama atau lihat-lihat daftar lengkap di bawah ini.
